# Todo App（Django）

##　概要
Djangoを使用したシンプルなTodo管理アプリです。  
タスクの追加・削除・完了管理ができます。

---

##　使用技術
- Python
- Django
- PostgreSQL
- Bootstrap 5
- HTML / CSS

---

##　機能
- タスクの追加
- タスク一覧表示
- 完了 / 未完了の切り替え
- タスク削除
- データベース連携（PostgreSQL）

---

##　工夫した点
- DjangoのMTVモデルを使ったCRUD処理の実装
- PostgreSQLとの連携によるデータ永続化
- Bootstrapを用いたUI改善で視認性を向上
- シンプルで使いやすい画面設計

---

- タスクの期限・優先度管理
- API化（React連携）
