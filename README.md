# Todoアプリ

## 概要
Djangoで作成したTodo管理アプリです。

## 使用技術
- Python
- Django
- PostgreSQL
- HTML/CSS
- Bootstrap

## 実装機能
- ユーザー登録・ログイン機能
- Todo登録
- Todo一覧表示
- Todo完了状態変更
- Todo削除
- ユーザーごとのTodo管理

## 工夫した点
- Django標準認証機能を利用しログイン機能を実装
- ユーザー単位でデータを管理できるよう設計
- Bootstrapを使用し画面デザインを改善
